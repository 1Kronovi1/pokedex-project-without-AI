# 🔴 Pokédex

> Explore, descubra e favorite os Pokémons que você mais gosta — tudo num lugar só.

---

## ✨ O que é isso?

Um projeto pessoal de Pokédex feito com HTML, CSS e JavaScript puro. A ideia foi simples: criar uma interface onde você consegue buscar qualquer Pokémon, navegar entre eles e salvar seus favoritos — sem precisar de nenhum framework.

Os dados vêm direto da [PokéAPI](https://pokeapi.co/), uma API pública e gratuita com informações de todos os Pokémons.

---

## 🚀 Funcionalidades

- 🔍 **Busca por nome** — digite o nome do Pokémon e veja ele aparecer na hora
- ⬅️ ➡️ **Navegação** — avance ou volte entre os Pokémons pelo ID
- ⭐ **Favoritar** — salve seus Pokémons favoritos com um clique na estrela
- 💾 **Persistência** — os favoritos ficam salvos no `localStorage`, então continuam lá mesmo depois de fechar o navegador
- 📱 **Responsivo** — funciona bem tanto no desktop quanto no celular

---

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- [PokéAPI](https://pokeapi.co/)
